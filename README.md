# Программа ConfigRedactor
## Что делает
Позволяет вам реализовать изменение XML-файла с определённой структурой.
## Как пользоваться?
В файле Parser/pathoffile.h в объявлении PATHOFFILE подставить нужный путь к файлу.

* Заходим в файл:
